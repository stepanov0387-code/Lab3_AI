# Лабораторная работа №3: Классификация типа поверхности мобильного робота

## Описание
В рамках работы реализован MLP-классификатор (многослойный перцептрон) для распознавания типа поверхности движения мобильного робота по сенсорным данным (угловые скорости, токи, виртуальные переменные V2, V3).  
Исследовано влияние различных методов предобработки данных:

- исходные данные
- специальная сортировка (перемежение классов)
- нормализация MinMaxScaler
- балансировка классов (SMOTE, ADASYN)

Целевой класс – **тип 2**.  

## Файлы
- `Lab3_p4_FINAL copy.ipynb` – основной Jupyter Notebook с реализацией всех экспериментов.
- `Отчет_Степанов_ЛР3.docx` – подробный отчёт с анализом результатов и выводами.
